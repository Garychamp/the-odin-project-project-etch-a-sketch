# Etch-A-Sketch

A digital recreation of the classic Etch-A-Sketch toy, built as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## 📋 Project Description

This project implements an interactive drawing application that mimics the experience of using an Etch-A-Sketch. Users can draw on a customizable grid, select different colors, and clear their artwork to start fresh.

## 🎨 Features

- Customizable grid size (adjustable dimensions)
- Color picker for drawing
- Hover effects to preview drawing areas
- Clear canvas functionality
- Responsive and intuitive user interface
- Real-time grid updates
- Smooth drawing experience

## 🛠️ Technologies Used

- **JavaScript** (55.6%) - Grid generation, event handling, and drawing logic
- **CSS** (25.4%) - Styling, grid layout, and interactive effects
- **HTML** (19%) - Structure and semantic markup

## 🚀 Getting Started

### Prerequisites
- A modern web browser

### Installation

1. Clone the repository:
   ```bash
git clone https://github.com/Garychamp/the-odin-project-project-etch-a-sketch.git
cd the-odin-project-project-etch-a-sketch
```

2. Open `index.html` in your web browser

## 📖 How to Use

1. **Adjust Grid Size**: Use the input field to set your desired grid dimensions (e.g., 16 for a 16x16 grid)
2. **Select Color**: Click the color picker to choose your drawing color
3. **Draw**: Hover over or click on grid squares to draw
4. **Clear Canvas**: Click the "Clear" button to reset the canvas and start over

## 🎯 Controls

| Action | Method |
|--------|--------|
| Draw | Hover over or click grid squares |
| Change Color | Use the color picker input |
| Adjust Grid | Enter new dimensions and update |
| Clear Canvas | Click the Clear button |
